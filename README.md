# 🔐 Password Generator

**Stop relying on weak passwords—generate strong, customizable ones instantly with a fast and modern React app.**

---

## 🚀 Features

- 🔑 Instant password generation
- 📏 Adjustable length (6–30 characters)
- 🔢 Optional numbers inclusion
- 🔣 Optional special characters
- 📋 One-click copy to clipboard
- ⚡ Fast and responsive UI

---

## 🧠 Key Concepts

This project demonstrates:

- React Functional Components
- useState for state management
- useMemo for derived state
- Controlled form inputs
- Clipboard API (`navigator.clipboard`)
- Clean UI logic without unnecessary effects

## ⚙️ Installation

### 1. Clone the repository
```
git clone https://github.com/aswanth-kt/Password_Generator.git

cd password-generator

npm install

npm run dev
```

## 🛠️ Usage
- Move the slider to set password length
- Toggle:
  - ✅ Numbers
  - ✅ Special Characters
- Click Copy to copy the generated password
- Password updates automatically based on selected options


## 🧪 Example Output
```
A7f@kL2#xP
```
